# Crazy Fool / Foolish Fool

Originally by Dee Dee Warwick. Bradley Nowell covered it acoustically as "Foolish Fool" on *Sublime Acoustic: Bradley Nowell & Friends* (1998). Danny's version is on the Cover Up album as "Foolish Fool" — same song, same lineage.
