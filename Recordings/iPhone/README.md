# Voice Memo Catalog

126 iPhone recordings across 4 folders — the raw creative pipeline behind the Equality Music catalog.

## Folder Structure

| Folder | Count | Description |
|--------|-------|-------------|
| **Final Cuts** | 48 | Polished voice recordings — best takes, ready for production |
| **Free Style (EP)** | 9 | Original EP recordings that became the '89 Vision self-titled album |
| **New Cuts** | 1 | Latest recordings not yet sorted |
| **Raw Cuts** | 68 | Working sessions — experiments, variants, freestyles, early ideas |

**Total runtime: ~3 hours 45 minutes**
